#### SkylineAnalysis 的属性说明如下

| 属性   | 说明              | 类型    | 默认值          |
| ------ | ----------------- | ------- | --------------- |
| lineColor | 天际线颜色 | Array | [1, 0, 0]           |
| lineWidth  | 天际线宽度      | number  | 1             |

#### 方法 : exportSkylineMap(options)
_输出天际线贴图_
| 参数名 | 说明 | 类型 | 默认值 |
| ------------- |---------- |-------------|--------- |
| `options`     | 参数选择项 | Object | 无 |
> returns :
base64: String
